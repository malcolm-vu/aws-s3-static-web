This is a sample index page for a S3-hosted static website with CloudFront distribution using Terraform

Some details of the architecture:

  • Terraform modules used for the different components
  • Cloudfront Origin Access Control (OAC) used to ensure secure access to S3 bucket
  • S3 server access and CloudFront logging
  • CloudWatch metric alarm triggering email via SNS notification for 404 errors

More information can be found in the Github page.
